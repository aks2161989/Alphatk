Option-click in titlebar

If you put the insertion point at an opening tag and hold down the option key while you click in the window's titlebar, a menu appears with attributes which you can insert. Select an attribute from the menu and it will be inserted in the tag. If you for example put the insertion point at a <STYLE> tag

<STYLE TYPE="text/css">

the menu to the right appears when you option-click in the titlebar. Note that the TYPE attribute does not appear in the menu as it is already used. Let us say you select MEDIA from the menu. This attribute is then inserted into the tag with the insertion point between the quotes to let you type the attribute value.

<STYLE TYPE="text/css" MEDIA="|"•>

But before you type the value of the MEDIA attribute, option-click again in the titlebar, and select TITLE from the menu shown to the left. This attribute is now inserted into the tag, but the insertion point is still between the quotes of the MEDIA attribute.

<STYLE TYPE="text/css" MEDIA="|" TITLE="•"•>

You can now continue to insert more attributes and when you are done type their values one after each other and use the tab key to jump between them.

Option-click in home page window

You can also option-click in the titlebar of home page windows to navigate between them. This is explained in detail in the section about home page windows.

Files in option-click menu

There is one more thing to note about the option-click menu. If there are fewer attributes than Min Items In Title Popup in the dialog box Config->Preferences->Interface Preferences->Window then Alpha adds the names of the files in the window's folder. Selecting a file from the menu opens the file.
