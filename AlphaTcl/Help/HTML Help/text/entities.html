Character entities

The Character Entities can be used to insert entities for characters, for example ü is &uuml;. All available entities are divided into four submenus. Picking a character from any of these menus is only useful if you only want a single character. If you write in a language which use a lot of these characters, type them in the document as usual and then use the character translation.

At the top of this menu there are some items, which can also be found in the submenus. These are there to be more easily accessible. You can edit the menu using Add... and Remove... to select which character you want to be able to access more quickly.

You can define key bindings for all character entities using Key Bindings in the Preferences menu. By default the following key bindings are defined:

shift-ctrl-opt-,
shift-ctrl-opt-.
shift-ctrl-opt-7
shift-ctrl-opt-space  
inserts &lt;
inserts &gt;
inserts &amp;
inserts &nbsp;
